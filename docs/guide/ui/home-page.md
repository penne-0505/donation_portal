# ホームページ - ユーザーガイド

> 本ページは、**ホームページ（LP）の表示・ナビゲーション・寄付フロー** の概要を説明します。

## 概要

**Donation Portal** のホームページは、Discord コミュニティメンバーが **最短で寄付を開始できる** ランディングページとして設計されています。

### 主な特徴

- **ヒーロー配置**: タイトル + リード + CTA ボタン 2 種
- **最小フッター**: プライバシー・運営者情報リンク
- **ヘッダー CTA**: 常時表示の「寄付する」主ボタン
- **バッジ 3 種**: 安心感・機能紹介を最小限に集約

## ナビゲーション

### ヘッダー要素

| 要素           | テキスト        | リンク先  | 用途           |
| -------------- | --------------- | --------- | -------------- |
| **Logo**       | Donation Portal | `/`       | ホーム戻却     |
| **補助リンク** | 支援者一覧      | `/donors` | 支援者情報表示 |
| **主 CTA**     | 寄付する        | `/donate` | 寄付フロー開始 |

### 主 CTA ボタンのデザイン

- **色**: 主色（通常は青系）
- **スタイル**: 塗り + 角丸 + フォーカスリング
- **サイズ**: `size="sm"`（ヘッダー用）
- **テキスト**: `寄付する`
- **アイコン**: なし
- **ホバー効果**: 背景色変化 + スケール変化

## ヒーロー セクション

### H1 テキスト

```
Discordコミュニティの運営を支える寄付
```

- **フォント**: Bold、段落下 `mt-4` で配置
- **PC**: `text-5xl`（5rem）
- **SP**: `text-4xl`（2.25rem）
- **特徴**: `text-balance` で行揃え最適化

### リードテキスト

```
透明性と感謝を大切に運営しています。
```

- **色**: `text-gray-600 dark:text-gray-400`（コントラスト 4.5:1）
- **フォント**: Regular、PC `text-lg` / SP `text-base`
- **位置**: H1 直下、`mt-2` で配置

### CTA ボタン 2 種

#### 主ボタン（塗り）

```
寄付する → (ArrowRight icon)
```

- **リンク**: `/donate`
- **スタイル**: `size="lg"`, `variant="primary"`, `px-8`
- **アクション**: クリック時 `donate_start` イベント発火
- **ARIA**: `aria-label="寄付をはじめる"`

#### 副ボタン（アウトライン）

```
支援者一覧
```

- **リンク**: `/donors`
- **スタイル**: `size="lg"`, `variant="outline"`
- **ARIA**: `aria-label="支援者一覧を表示"`

**配置**:

- PC: 横並び（`flex-row`, `gap-3`）
- SP: 縦積み（`flex-col`, `gap-3`）

### バッジ 3 種

各バッジは、ヒーロー下部に **左中右** で配置。

| #   | テキスト           | 絵文字 | 意図               |
| --- | ------------------ | ------ | ------------------ |
| 1   | Stripeで安全決済   | 🔒     | 決済の安全性       |
| 2   | OAuthで同意管理    | ✅     | 同意管理の透明性   |
| 3   | 支援者リストを公開 | 📋     | 支援者情報の透明性 |

**バッジデザイン**:

- **形状**: `rounded-full`（完全丸形）
- **枠線**: `border border-border/60`
- **背景**: `bg-white/5` + `backdrop-blur`
- **パディング**: `px-4 py-2`
- **テキストサイズ**: SP `text-xs` / PC `md:text-sm`
- **折返し**: なし（`nowrap` 意図）

**配置**:

- `flex flex-wrap justify-center gap-3 pt-4 md:pt-6`

## 実装メモ

- ホームページのヒーロー／CTA セクションは `components/pages/home-page.tsx` に集約され、`SectionHeading`（`components/ui/section-heading.tsx`）と `Card` の `surface`/`padding` props で視覚トーンを統一しています。
- ヘッダー／フッターを含む共通レイアウトは `app/(main)/layout.tsx` → `components/app-shell.tsx` で提供され、`HeroProvider` が LP ボタンのデエンファシス制御を担います。
- 新しい UI foundation では `useHeroContext` と `useDonationFlow` の副作用を分離しているため、ホームページから寄付ボタンをクリックした際の `gtag('event','donate_start')` は `DonatePage` 側に一本化されています。
- Tailwind v4 inline テーマの `glass-*`/`glow-*` クラスを直接参照せず、`Button` や `Card` の variants を経由することでデザインの一貫性を保ちます。

## ユースケース

### ユースケース 1: 新規メンバーが寄付を開始したい

1. Discord リンクから LP へ到着
2. ヘッダー「寄付する」またはヒーロー「寄付する」をクリック
3. `/donate` ページで Discord ログイン → 寄付額選択 → Stripe Checkout
4. `/thanks` で完了

**最短時間**: クリック含め **30 秒**

### ユースケース 2: 支援者が自分の寄付を確認したい

1. LP から「支援者一覧」をクリック
2. `/donors` で支援者リストを表示
3. 自分の表示名が同意済みの場合、一覧に表示

## アクセシビリティ

### キーボードナビゲーション

| キー              | 動作                   |
| ----------------- | ---------------------- |
| `Tab`             | 要素間をフォーカス移動 |
| `Enter` / `Space` | ボタン/リンククリック  |
| `Shift+Tab`       | 逆方向フォーカス移動   |

### フォーカス表示

- **方法**: `focus-visible:ring-2`（青色枠）
- **対象**: ボタン・リンク・フォーム要素
- **説明**: 目で見てわかる、かつ背景と十分コントラスト

### スクリーンリーダー対応

- **見出しマークアップ**: H1 は `<h1>` タグ
- **ボタンラベル**: `aria-label="寄付をはじめる"`
- **装飾要素**: `aria-hidden` で画面読み上げ非表示化
- **リンク**: テキストは意味のある説明

## 計測・アナリティクス

### イベント一覧

本ページから発火する計測イベント：

| イベント名         | トリガー                       | 値  |
| ------------------ | ------------------------------ | --- |
| `hero_cta_click`   | ヘッダー「寄付する」クリック   | 1   |
| `donor_list_click` | ヘッダー「支援者一覧」クリック | 1   |
| `donate_start`     | ヒーロー「寄付する」クリック   | 1   |

### 計測方法

```typescript
// イベント発火例
if (typeof window !== 'undefined' && (window as any).gtag) {
  (window as any).gtag('event', 'hero_cta_click');
}
```

### 主要指標

- **LP 訪問数**: `page_view`
- **hero_cta_click CTR**: `hero_cta_click / LP 訪問数`
- **寄付フロー移行率**: `donate_start / hero_cta_click`

## トラブルシューティング

### Q: CTA ボタンが見つからない

**A**: ヘッダー右側にピンク色の「寄付する」ボタンがあります。モバイルでも同位置（右上）に表示されます。

### Q: 支援者一覧に名前が表示されない

**A**: 以下を確認してください：

1. `/donate` で同意チェックボックスが ON になっているか
2. Stripe 寄付が完了しているか
3. 同意後 5 分程度で反映（キャッシュ遅延）

### Q: モバイルでテキストが読みづらい

**A**: フォントサイズは自動調整されます。ブラウザのズーム機能で拡大できます（`Ctrl+` or `Cmd+`）
